PingMe

Connect, Chat, and Collaborate in Real-Time

PingMe is a modern, feature-rich chat application designed to bring people together through seamless communication. Whether you're connecting with friends, collaborating with teams, or building communities, PingMe provides the perfect platform for all your messaging needs.

Key Features

Public & Private Rooms

Create public rooms for community discussions or private rooms for intimate conversations with your team or friends.

Private Chat

Enjoy one-on-one conversations with read receipts, message editing, and file sharing capabilities.

Rich Profiles

Create detailed profiles with photos, bio, date of birth, and location. Express yourself fully!

Friend System

Send, accept, and manage friend requests. Build your network and stay connected with people you care about.

Media Sharing

Share images, documents, and files seamlessly within your chats and rooms.

Secure & Verified

Email-based OTP verification for registration and password reset ensures your account security.

Technology Stack

PingMe is built with modern, reliable technologies to ensure the best user experience and performance.

Django

Robust Python web framework powering the backend with ORM, authentication, and security features.

Bootstrap

Modern CSS framework ensuring responsive design and beautiful user interface across all devices.

AJAX

Dynamic, real-time interactions without page reloads for a smooth, app-like experience.
